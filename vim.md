## Vim snippet commands : 

> This is a list of commands we can use in vi.

### openning a File : 
```shell
vim file_name
```

### Saving a File : 
```shell
:wq
```


### Cutting a File : 
```shell
dd
```


### Pasting a File : 
```shell
p
```


### Searching a File : 
```shell
/
```


### Undoing a File : 
```shell
u
```


### Quitting a File : 
```shell
:q!
```


### inserting a File : 
```shell
i
```

### end of the line in a File : 
```shell
$
```

### start of the line in a File : 
```shell
0
```
